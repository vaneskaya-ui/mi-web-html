## 📝 Descripción

Este proyecto consiste en la creación de un sitio web utilizando HTML5 y CSS3.

Incluye:

* Página de inicio (index.html)
* Galería de productos
* Formulario de presupuesto con validación HTML5
* Página de contacto con mapa e información

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
